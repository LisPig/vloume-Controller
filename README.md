chrome浏览器插件

功能：整合控制音视频网站的音量大小，双向调节音量。

未完善：目前不能支持每个音视频网站，但也满足大部分头部网站。有些网站不是普通的播放标签编写，所以代码逻辑捕捉不到播放元素。这是待完善的一项。

项目90%代码由chatGPT提供，UI由figma绘制。
